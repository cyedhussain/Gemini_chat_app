# 🌸 Gemini AI Chat App

A modern and beautiful AI chatbot application built with Flutter using the Gemini API.  

This app features a clean chat interface with animated message bubbles, smooth UI, and a premium user experience inspired by modern AI applications.

---

# ✨ Features

- 🤖 Gemini AI Integration  
- 💬 Real-time AI Chat  
- 🎨 Modern & Attractive UI  
- 🌸 Pink AI Response Bubbles  
- 🔵 Blue User Message Bubbles  
- ⚡ Fast API Response Handling  
- 📱 Responsive Flutter Design  
- 🧠 Clean & Simple Architecture  
- 🌙 Smooth User Experience

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/gemini_chat_app.git
```

## 2️⃣ Open Project

```bash
cd gemini_chat_app
```

## 3️⃣ Install Dependencies

```bash
flutter pub get
```

## 4️⃣ Add Gemini API Key

Go to your `HomeScreen` file and add your API key:

```dart
'x-goog-api-key': 'YOUR_API_KEY',
```

---

# ▶️ Run the App

```bash
flutter run
```
